# Monascope

A premium futuristic AI-powered web platform built using React and Tailwind CSS.

## Overview

Monascope is designed as a world-class SaaS platform inspired by:

* Apple
* Google
* OpenAI
* Linear
* Stripe

The goal is to deliver:

* Premium UI/UX
* Enterprise trust
* Modern responsive design
* Intelligent product experience

---

## Tech Stack

* React
* Vite
* Tailwind CSS
* React Router DOM
* JavaScript (ES6+)

## Installation

Clone repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
npm run dev
```
